Unit Converter Project

A solo project created as part of my front-end dev course on Scrimba.

Project features:

  • Automatically performs metric/imperial unit conversions
  • Converts Meters to Feet
  • Converts Litres to Gallons
  • Converts Kilograms to Pounds

Developed by Obinna Osisiogu
